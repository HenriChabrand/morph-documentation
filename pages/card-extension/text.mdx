# Text

Text content is used frequently in card development to display information to users. Our API provides the `CardText` class to help you create and format text content in your cards. The `CardText` method enables you to add informative text fields in your card components.

## API Reference

### CardText

Component to add textual data in your card with optional link.

Example:

```tsx
const newCard = cardBuilder.newCard("User Profile");

newCard.newText("Name", "John Doe");
newCard.newText("Handle", "@john", "https://abc.com/john");
newCard.newText("Role", "Product Manager");
```

## Props

| Prop    | Description                     | Type   | Default |
| ------- | ------------------------------- | ------ | ------- |
| label\* | The label for the text field.   | string | -       |
| value\* | The text value to be displayed. | string | -       |
| link    | Optional URL to link the text.  | string | -       |

`*` indicates required fields
